# Personal Blog Website

This is the project of the Udacity Nanodegree Program <em>Front End Developer Nanodegree Program</em>.
It is assigned in the second unit of the course: <em>CSS, Website Layout, Website</em>.

The task required is to design a personal blog following the project rubric specified.

My blog consist on a web where I share information about Machine Learning projects that I have developed or plan to design. I include theoretical concepts that help to understand the technique used for each one of them, as well as the link of the corresponding github repository where technical information about the functionality is provided.
